# Hangman Word Game

- **Run the game:**
  - Open terminal in this folder
  - Run: `python3 main.py`

- **Play instructions:**
  - Choose menu options by number
  - Select difficulty, guess letters, view/save/load records

- **Test the code:**
  - Run: `python3 -m unittest test_main.py`
  - Tests cover input validation and game logic

- **Files:**
  - `main.py`: Game logic and menu
  - `test_main.py`: Unit tests
  - `JOURNAL.md`: Session log
  - `save.txt`: Saved records

- **Requirements:**
  - Python 3.7+
  - No external dependencies

- **Notes:**
  - No `while True` loops used
  - Game records are saved/loaded as plain text
  - ASCII art for hangman
